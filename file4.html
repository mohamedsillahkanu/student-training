How It Works - Self-Contained Version

CSV Processing Features:

  • No External Dependencies: All processing done with native JavaScript
  • All Columns Preserved: Combines ALL columns from all files
  • Data Cleaning: Removes empty rows and cleans data
  • Source Tracking: Adds 'source_file' column
  • Self-Contained: Works completely offline

Iframe-Friendly Design:

  • Scoped CSS classes to avoid conflicts
  • No global style overrides
  • Responsive design for small containers
  • Self-contained widget approach
  • Works in any container size
Note: This version only supports CSV files and is designed to work safely in iframes and embedded containers without affecting parent page styles.

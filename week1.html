Step 1: Basic Summary Statistics

ğŸ” Explanation:

ğŸ” Explanation:

2ï¸âƒ£ Handling Missing Values in Malaria Data

Step 1: Identifying Missing Values

# Check for missing values in the dataset
missing_values <- colSums(is.na(malaria_data))
print(missing_values)
# Calculate the percentage of missing values
missing_percentage <- colMeans(is.na(malaria_data)) * 100
print(missing_percentage)
ğŸ” Explanation:

âœ” is.na() identifies missing values in the dataset
âœ” colSums() sums the missing values in each column
âœ” colMeans() calculates the proportion of missing values in each column

Step 2: Removing Rows with Missing Values

# Remove rows with any missing values
complete_data <- na.omit(malaria_data)
# Check how many rows were removed
original_rows <- nrow(malaria_data)
complete_rows <- nrow(complete_data)
removed_rows <- original_rows - complete_rows
print(paste("Removed", removed_rows, "rows with missing data"))
ğŸ” Explanation:

âœ” na.omit() removes any row that contains at least one missing value
âœ” This is a simple approach but may remove too much data if missing values are common

Step 3: Replacing Missing Values with tidyr

# Load tidyr package
library(tidyr)
# Replace missing values in 'cases' with the mean
malaria_data_imputed <- malaria_data %>%
mutate(cases = replace_na(cases, mean(cases, na.rm = TRUE)))
# Replace missing values in 'severity' with "Unknown"
malaria_data_imputed <- malaria_data_imputed %>%
mutate(severity = replace_na(severity, "Unknown"))
ğŸ” Explanation:

âœ” tidyr::replace_na() replaces NA values with specified replacements
âœ” For numeric columns, we often replace with mean, median, or zero
âœ” For categorical columns, we may replace with a new category like "Unknown"

ğŸ” Explanation:

âœ” naniar package provides specialized tools for exploring missing data
âœ” gg_miss_var() creates a plot showing the count of missing values by variable
âœ” vis_miss() creates a matrix visualization of missing values throughout the dataset

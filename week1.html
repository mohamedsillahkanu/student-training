2Ô∏è‚É£ Basic R Syntax

Step 1: Assigning Variables

cases <- 150 # Number of malaria cases
fatality_rate <- 4.2 # Fatality rate in percentage
region <- "West" # Region name (text)
outbreak <- TRUE # Logical value (Boolean)
üîç Explanation:

‚úî cases <- 150 assigns 150 to cases
‚úî fatality_rate <- 4.2 stores 4.2 in fatality_rate
‚úî region <- "West" stores "West" as a text string
‚úî outbreak <- TRUE stores a Boolean value (TRUE/FALSE)

Step 2: Checking Data Types

class(cases) # Output: "numeric"
class(region) # Output: "character"
class(outbreak) # Output: "logical"
üîç Explanation:

‚úî class() function shows the data type of each variable
‚úî "numeric" means numbers
‚úî "character" means text
‚úî "logical" means TRUE or FALSE

Step 3: Basic Operators

total_cases <- cases + 50 # Adding 50 more cases
fatalities <- cases * (fatality_rate / 100) # Calculating fatalities
recovered <- cases - fatalities # Subtracting fatalities from total cases
üîç Explanation:

‚úî cases + 50 adds 50 cases
‚úî cases * (fatality_rate / 100) converts percentage to decimal and calculates fatalities
‚úî cases - fatalities finds the number of recovered patients

4Ô∏è‚É£ Reading Malaria Datasets (CSV, Excel) in R

Why Read External Data?

üìå Malaria data is stored in CSV (Comma-Separated Values) or Excel files.
üìå We use readr and readxl to load these files into R.

Step 1: Install Required Packages

install.packages("readr") # Install readr package for CSV files
install.packages("readxl") # Install readxl package for Excel files
üîç Explanation:

‚úî install.packages("readr") installs readr (for CSV files)
‚úî install.packages("readxl") installs readxl (for Excel files)

üìÇ 4.1 Reading CSV Files with readr

library(readr) # Load readr package

# Read malaria data from a CSV file
malaria_data <- read_csv("malaria_cases.csv")

# View first few rows
head(malaria_data)
üîç Explanation:

‚úî library(readr) loads the readr package
‚úî read_csv("malaria_cases.csv") reads the CSV file into a dataframe
‚úî malaria_data stores the imported dataset
‚úî head(malaria_data) shows the first 6 rows of the dataset

üìÇ 4.2 Reading Excel Files with readxl

library(readxl) # Load readxl package

# Read malaria data from an Excel file
malaria_data_excel <- read_excel("malaria_cases.xlsx", sheet = 1)

# View first few rows
head(malaria_data_excel)
üîç Explanation:

‚úî library(readxl) loads the readxl package
‚úî read_excel("malaria_cases.xlsx", sheet = 1) reads Sheet 1 of the Excel file
‚úî malaria_data_excel stores the imported dataset
‚úî head(malaria_data_excel) shows the first 6 rows

üìå 4.3 Writing Data to CSV and Excel

Saving as CSV
write_csv(malaria_data, "cleaned_malaria_data.csv")
üîç Explanation:

‚úî write_csv(malaria_data, "cleaned_malaria_data.csv") saves malaria_data as a CSV file

Saving as Excel
install.packages("writexl") # Install writexl package
library(writexl) # Load writexl

write_xlsx(malaria_data, "cleaned_malaria_data.xlsx")
üîç Explanation:

‚úî install.packages("writexl") installs writexl for writing Excel files
‚úî library(writexl) loads the writexl package
‚úî write_xlsx(malaria_data, "cleaned_malaria_data.xlsx") saves data as an Excel file
